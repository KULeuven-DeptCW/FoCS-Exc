FoCS-Exc
========

Exercise sessions of "Fundamentals of Computer Science".

Content
-------
Exercise sessions:

 1. Finite state machines and regular expressions
 2. Context-free grammars
 3. Turing machines
 4. Graph theory
 5. Computational complexity.
